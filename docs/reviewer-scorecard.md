# Nevora Codex v1.0 Reviewer Scorecard

This scorecard gives reviewers a simple way to summarize release readiness across the seven v1.0 review gates.

It is not a replacement for detailed comments, issues, or pull requests. It is a final summary aid.

## Rating scale

Use the following scale for each section:

| Score | Meaning |
| --- | --- |
| 0 | Not reviewed |
| 1 | Major concerns / release blocker likely |
| 2 | Needs revision before release |
| 3 | Acceptable with minor edits |
| 4 | Strong / release-ready |
| 5 | Excellent / unusually clear and safe |
